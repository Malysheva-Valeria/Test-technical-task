Hospital Management System

Overview

A comprehensive hospital management module for Odoo that helps healthcare facilities manage doctors, patients, diseases, and patient visits efficiently.

Features

Doctor Management

  • Complete doctor profiles with specialties
  • Mentor-intern relationships
  • Patient assignment tracking
  • Visit history management

Patient Management

  • Comprehensive patient records
  • Automatic age calculation
  • Disease history tracking
  • Assigned doctor monitoring

Disease Management

  • Disease type categorization
  • ICD-10 code support
  • Patient-disease relationships
  • Detailed descriptions

Visit Tracking

  • Complete visit documentation
  • Diagnosis and prescription records
  • Visit status workflow
  • Historical visit data

Installation

  1. Download or clone this module to your Odoo addons directory
  2. Update the apps list in Odoo
  3. Search for "Hospital Management" and click Install

Configuration

After installation, the module provides demo data including:

  • 3 sample disease types
  • 3 sample doctors
  • 3 sample patients

Access the Hospital menu to start managing your healthcare facility.
